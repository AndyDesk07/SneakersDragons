¡Bienvenido a **SNEAKERS DRAGONS**! Este es un sistema de gestión de inventario premium diseñado para tiendas de calzado deportivo. El proyecto permite administrar el stock de modelos icónicos con una interfaz futurista y de alto rendimiento.

## 🚀 Características Principales

* **Gestión de Inventario (CRUD):** Permite Agregar, Visualizar, Editar y Eliminar productos del stock en tiempo real.
* **Interfaz Premium:** Diseño oscuro con estética neón "Cyberpunk" y animaciones fluidas.
* **Selección Inteligente:** Input híbrido que permite escribir modelos manualmente o seleccionar de una lista de sugerencias predefinida (Nike Shox, Jordan 1, Jordan 5, etc.).
* **Visualización Dinámica:** Las tarjetas de productos detectan automáticamente el modelo ingresado y muestran la imagen correspondiente.
* **Notificaciones Visuales:** Sistema de alertas flotantes para confirmar acciones del usuario.
* **Lluvia Visual:** Efecto de fondo con partículas (canvas) para una experiencia inmersiva.

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica del sistema.
* **CSS3:** Estilos personalizados, animaciones complejas (keyframe effects) y diseño responsivo.
* **JavaScript (Vanilla):** Lógica de negocio, manipulación del DOM y renderizado dinámico.
