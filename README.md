# Tienda-de-zapatillas
# 👟 SneakerDRAGON - Catálogo de Zapatillas
**Desarrollado por:** Christian Jimenez

Proyecto de Front End diseñado para una tienda de calzado deportivo, enfocado en estructuras semánticas modernas, diseño responsivo y buenas prácticas de desarrollo.

---

## 📋 Cumplimiento de Pauta de Evaluación

### 1.1.1 Identificación de elementos HTML (Semántica)
Se implementó una estructura basada en etiquetas semánticas de HTML5 para mejorar el SEO y la accesibilidad:
* **`<header>`**: Contenedor de navegación y logo.
* **`<main>`**: Núcleo del contenido del catálogo.
* **`<section>`**: División temática (Hero y Galería de productos).
* **`<article>`**: Cada zapatilla está encapsulada como una entidad independiente.
* **`<footer>`**: Información de autoría y derechos.

### 1.1.2 Construcción y Buenas Prácticas (IA)
Código generado y optimizado con asistencia de IA para asegurar:
* **Indentación limpia:** Código jerarquizado y fácil de leer.
* **Naming Semántico:** Clases de CSS descriptivas.
* **Optimización:** Código ligero y libre de etiquetas innecesarias.

### 1.1.3 Propiedades de CSS3 y Accesibilidad
Diseño moderno y adaptable cumpliendo con:
* **Layout:** Uso de **CSS Grid** para la grilla de productos y **Flexbox** para la barra de navegación.
* **Responsividad:** Implementación de `@media queries` para visualización en móviles.
* **Accesibilidad:** Contraste de colores adecuado y uso de atributos `alt` en todas las imágenes.

### 1.1.4 Control de Versiones
Gestión profesional del proyecto mediante:
* **GitHub:** Repositorio en la nube como política de respaldo.
* **Commits Semánticos:** Mensajes claros que describen cada avance del proyecto.
